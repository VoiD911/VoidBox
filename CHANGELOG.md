# Changelog

## v1.5.0 (2026-03-06)

- New: separate tank frame — optional dedicated panel showing only tank(s), movable independently
  - Enable via checkbox "Show separate tank frame" in Appearance tab
  - Auto-hides when no tanks are in the group
  - Supports drag to reposition, lock/unlock, profile save
- Change: scale sliders now use 5% steps instead of 10% for finer control
- Added German translation for tank frame option

## v1.4.1 (2026-03-03)

- New: minimap button (LibDBIcon) — left-click to toggle frames visibility, right-click to open config
- New: `/vb minimap` slash command to show/hide the minimap button
- New: "Show minimap button" checkbox in Appearance tab
- Added: LibStub, CallbackHandler-1.0, LibDataBroker-1.1, LibDBIcon-1.0 embedded libraries

## v1.4.0 (2026-02-22)

- New: universal "Press to Bind" system — supports keyboard keys, mouse clicks, scroll wheel, and MMO mouse buttons
- New: automatic migration from v1 binding format to v2

## v1.3.1 (2026-02-17)

- Fix: stack count badge not displaying (secret string from string.format)

## v1.3.0 (2026-02-17)

- New: percentage-based scaling system (50%-150%, live preview)
- New: debuff icons row with stack count badges
- New: HOT/shield icons row (player-cast only) with "+N" indicator
- New: tooltip repositioned to bottom-right of screen

## v1.2.2 (2026-02-16)

- New: real role icons (atlas), dynamic name truncation

## v1.2.0 (2026-02-15)

- New: mousewheel bindings, macro drag & drop, profile system

## v1.1.0 (2026-02-14)

- New: orientation, role order, frames per row options

## v1.0.0 (2026-02-10)

- Initial release
